### 報告書CF-14 (エンド拠点 第三セクタ) 
コーラスフルーツをより育てやすく、美しい植物へと改良する計画は、完全な失敗に終わった。  
我々は、わずかに成長の制御と色彩の調整を試みただけだったはずだ。  
だが、それによって得られた結果は、制御不能な増殖、異常な自己再生、そして終わりなき成長だった。

最初の兆候が現れたのは、エンド拠点で栽培していた試作品が「枯れず」に再び開花した時だ。  
それからわずか数日で、島全体は紫に染まり、既存の動植物はすべて飲み込まれた。  
観測装置の映像は、今もなお微かに動いている。

さらなるコーラスの浸食を防ぐため、エンドはもはや放棄するほかないだろう。  
我々は「過剰成長プログラム」を実行し、コーラスの自己崩壊に賭けようと思う。  

― *Dr. Mega Chorus*
